## Extensão para leitores de telas:

Ferramenta que ajuda em testes, evitar redundancias e leituras de coisas que voce nao espera
Tem uma extensão no google chrome Screen Reader


## Role

Define algumas semanticas a mais para os nossos elementos html

Exemplo: Role="dialog" é uma semantica para se colocar em modais personalizados para avisar para os leitores
de tela que algo abriu na tela, o dialog já é um componente do html
